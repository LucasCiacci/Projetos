2.2 Mediana e Moda:

Conteúdo Detalhado:

A mediana é útil em conjuntos de dados com valores extremos, pois não é afetada por eles como a média. Para calculá-la, os dados são ordenados e o valor centralé escolhido. Se houver um número ímpar de dados, a mediana é o valor do meio; se for par, é a média dos dois valores centrais.

A moda , por sua vez, é o valor que mais se repete em um conjunto de dados. Ela é especialmente útil em dados qualitativos ou categóricos, onde a média e a mediana podem não fazer sentido.

Exemplo:

Exemplo de Mediana: Considere o conjunto de números [4, 8, 15, 16, 23, 42, 50]. Para encontrar a mediana, ordenamos os números e pegamos o valor do meio. Neste caso, a mediana é 16 (o quarto número de uma lista de sete).

Exemplo de Moda: No conjunto de dados [5, 7, 9, 9, 11, 11, 11, 13], o valor que mais se repete é 11, então a moda é 11.
